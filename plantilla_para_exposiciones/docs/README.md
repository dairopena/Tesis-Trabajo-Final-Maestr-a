## 🎯 ¿Por dónde empezar?

### Si eres nuevo en Beamer:

1. Lee el [README.md](../README.md) principal
2. Consulta [REFERENCIA_RAPIDA.md](REFERENCIA_RAPIDA.md) para comandos básicos
3. Revisa [EJEMPLOS_USO.md](EJEMPLOS_USO.md) cuando necesites hacer algo específico

### Si quieres personalizar:

1. Lee [GUIA_FUENTES.md](GUIA_FUENTES.md) para cambiar la tipografía
2. Edita `config/preambulo.tex` con tus preferencias
3. Usa [EJEMPLOS_USO.md](EJEMPLOS_USO.md) como referencia para tu contenido

### Si necesitas ayuda rápida:

1. Ve directo a [REFERENCIA_RAPIDA.md](REFERENCIA_RAPIDA.md)
2. Busca el comando que necesitas
3. Copia y adapta a tu presentación
